This program implements the Alpha–Beta Pruning algorithm, an optimization of the Minimax algorithm used in decision-making and game-tree search. The program evaluates possible moves while pruning unnecessary branches to improve efficiency and reduce computation time.

To run the program:

Ensure Python 3 is installed on your system.
Open a terminal or command prompt in the folder containing the program file.

Run the program using the command:

python alpha_beta.py

(Replace alpha_beta.py with the actual filename if different.)

Follow any prompts displayed in the terminal if input is required.

The program outputs the optimal evaluation value calculated using alpha–beta pruning.
